# 📌 Project Overview

This project is an end-to-end **Mobile Sales Analytics Dashboard** developed using **Power BI** to analyze retail mobile sales data and generate actionable business insights.

The dashboard enables users to:

- Track overall sales performance
- Compare brand and model-wise sales
- Monitor monthly and daily trends
- Analyze customer ratings
- Compare Year-over-Year (YoY) performance
- Evaluate Month-to-Date (MTD) sales growth

---

# 📄 Dashboard Pages

| Dashboard Page | Description |
|----------------|-------------|

| 📅 MTD Sale Dashboard | Month-to-date cumulative sales trend analysis |
| 📆 Same Period Last Year | YoY comparison with previous year performance |
